### Support-Ende
Aufgrund von PSD2-Vorgaben hat Wise (ehemals TransferWise) leider die Abrufmöglichkeit von Umsatzbuchungen über die Schnittstelle eingestellt. Unter [https://wise.com/settings/public-keys](https://wise.com/settings/public-keys) findet sich hierzu auch ein entsprechender Hinweis.

Das Hibiscus-Plugin "hibiscus.transferwise" kann daher leider nicht mehr verwendet werden. Im [Blog](https://www.willuhn.de/blog/index.php?/archives/894-Support-Ende-fuer-hibiscus.transferwise.html) finden sich weitere Infos.
